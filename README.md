<h1>🪪Um sistema automatizado de uma rede de lojas de produtos.</h1>

<h2>💻Tecnologia:</h2>

- Linguagem C

<h2>🚀Funcionalidades do sistema:</h2>
<ul>
    <li> Criar_loja = cria e inicializa um nó Loja com cópia segura do nome.</li>
    <li> Inserir_inicio = insere uma loja no início da rota em tempo.</li>
    <li> Inserir_na_posicao = insere uma loja em uma posição específica; se a posição for maior que o tamanho, insere no fim.</li>
    <li> Inserir_fim = adiciona uma loja ao final da rota.</li>
    <li> Tamanho_rota = retorna o número de lojas na rota.</li>
    <li> Imprimir_rota = imprime os nomes das lojas na ordem da lista.</li>
    <li> Liberar_rota = libera toda a memória alocada e zera o ponteiro inicial.</li>
</ul>
